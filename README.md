tufo
====

Odor unknown

Pre-requisites
--------------

1.  **git** (http://git-scm.org)
2.  **node** + npm (http://nodejs.org)
3.  **bower** (`npm install -g bower`)

Install
-------

    git clone https://github.com/pfraces-wip/tufo.git
    cd tufo
    npm install && bower install

Usage
-----

    cd /path/to/tufo
    grunt

Browse to http://localhost:3000
